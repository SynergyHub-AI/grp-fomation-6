# **App Name**: SynergyHub AI

## Core Features:

- User Profile Management: Create and manage user profiles with name, bio, skills, interests, availability, and experience level.
- Skill Verification System: Verify user skills through GitHub link, portfolio, or live project link, with a 'Verified by Link' badge or a 'Self-Declared' label.
- AI-Powered Project Matching: Recommend suitable projects to users based on their skills, interests, and availability, displaying the match percentage and an Expert/Learner tag.
- Project Creation and Management: Create projects with title, type, description, required skills, required roles, team size, time commitment and optional location.
- AI Candidate Recommendation Tool: Recommend suitable candidates for projects based on skill match, interest match, availability, and learning intent. The tool will consider all this to make an optimal suggestion.
- Team Collaboration Space: Enable team collaboration with group chat, a project requirements board (problem statement, goals, tech stack, deliverables, timeline) and the assignment of roles.
- Learner Assignment: Facilitate an 'I Want to Learn' mode where users are only assigned to non-critical roles within a project.
- Firebase Integration: Store users, project and all the other related information to Firebase.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a sense of collaboration, intelligence, and focus.
- Background color: Light grey (#E0E0E0) for a clean, professional backdrop.
- Accent color: Purple (#800080) to create a modern feel for calls to action and key features.
- Headline font: 'Space Grotesk', sans-serif, for a contemporary, computerized look. Use 'Inter' for body text.
- Use flat, minimalist icons with a focus on clarity and ease of understanding, to complement the indigo color palette.
- Maintain a clean, organized layout with clear section divisions for each module to ensure an intuitive user experience.
- Use subtle transitions and loading animations to enhance user engagement without overwhelming the user interface.
